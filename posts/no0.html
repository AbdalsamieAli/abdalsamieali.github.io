  • Openloop Control


    2026-01-19

    Fundamentally, there are two types of control loop: open-loop control (feedforward), and closed-loop control (feedback).

    Open loop System:


    An open-loop control system is a type of control system where the control action is independent of the output of the system, meaning there is no feedback path to monitor or correct the output.This system operates based on a predetermined input without measuring the actual output, relying instead on the assumption that the system will perform as expected under normal conditions.
